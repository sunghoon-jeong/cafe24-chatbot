# 카페24 CS 챗봇 가이드

## 1. 이게 뭐야?

카페24 쇼핑몰에 붙이는 **고객 상담 챗봇**입니다.
자주 묻는 질문은 **자동응답(무료)**, 복잡한 질문은 **AI(유료)**가 대답합니다.

---

## 2. 어떻게 작동해?

```
고객이 질문 입력
       ↓
  키워드 매칭 확인
       ↓
  ┌─── 매칭됨 ──→ 템플릿 자동응답 (무료, 즉시)
  │
  └─── 매칭 안됨 → Claude AI 호출 (유료, 1~3초)
```

### 템플릿 자동응답 (무료)
아래 키워드가 포함되면 미리 만들어둔 답변이 바로 나감:

| 질문 종류 | 매칭되는 키워드 예시 |
|-----------|---------------------|
| 배송 조회 | 배송조회, 송장, 택배조회 |
| 배송 기간 | 며칠, 언제 도착, 배송기간 |
| 배송비 | 배송비, 무료배송 |
| 반품 | 반품, 반품신청 |
| 교환 | 교환, 사이즈교환 |
| 환불 | 환불, 결제취소, 주문취소 |
| 영업시간 | 영업시간, 몇시, 점심시간 |
| 연락처 | 전화번호, 주소, 위치 |
| 인사 | 안녕, 하이 |

### AI 응답 (유료)
위 키워드에 안 걸리는 질문만 Claude AI가 처리.
- 모델: Claude Haiku (가장 저렴한 모델)
- 1건당 약 1~2원
- 하루 100건이면 월 3,000~6,000원 수준

## 3. 파일 구조 (뭐가 뭔지)

```
챗봇 만들자/
├── app.py              ← 서버 메인 코드 (건드릴 일 거의 없음)
├── store_config.py     ← ⭐ 쇼핑몰 정보 + 자동응답 템플릿 (여기서 수정!)
├── requirements.txt    ← 필요한 프로그램 목록 (건드릴 일 없음)
├── runtime.txt         ← Python 버전 (건드릴 일 없음)
├── Procfile            ← 서버 실행 방법 (건드릴 일 없음)
├── railway.toml        ← Railway 배포 설정 (건드릴 일 없음)
├── .env.example        ← 환경변수 예시
├── .gitignore          ← Git에 안 올릴 파일 목록
└── static/
    ├── index.html      ← 챗봇 화면 디자인
    └── widget.js       ← 카페24에 삽입할 스크립트
```

**일반적으로 수정할 파일은 `store_config.py` 하나뿐입니다!**

## 4. 서버 구조

```
카페24 쇼핑몰 (내 쇼핑몰)
    ↓ widget.js가 API 호출
Railway 서버 (챗봇 백엔드)
    ↓ 필요할 때만
Anthropic API (Claude AI)
```

| 구성 | 서비스 | 비용 |
|------|--------|------|
| 쇼핑몰 | 카페24 무료 호스팅 | 무료 |
| 챗봇 서버 | Railway | 무료~월 $5 |
| AI | Anthropic Claude Haiku | 사용한 만큼 (매우 저렴) |

## 5. 유지보수 방법

### 자동응답 수정/추가하기
`store_config.py` 파일에서 `TEMPLATES` 부분을 수정하면 됨.

예시) "적립금" 관련 자동응답 추가:
```python
{
    "name": "적립금",
    "keywords": ["적립금", "포인트", "마일리지"],
    "answer": (
        "💰 적립금 안내드립니다!\n\n"
        "• 구매 금액의 1% 적립\n"
        "• 1,000원 이상부터 사용 가능\n"
    ),
},
```

### 쇼핑몰 정보 수정하기
`store_config.py` 파일 상단의 `STORE_INFO`에서 수정:
- 상호명, 전화번호, 이메일, 주소
- 배송비, 배송 기간
- 반품/교환/환불 정책

### 수정 후 서버에 반영하기
코드 수정 후 아래 명령어 실행:
```
cd ~/Desktop/챗봇\ 만들자
git add -A
git commit -m "내용 수정"
git push
railway up --detach
```
또는 Claude Code에서 "수정한 거 서버에 올려줘" 하면 됨.

## 6. 문제가 생겼을 때

### 챗봇이 응답을 안 해요
1. Railway 서버 확인: https://railway.com 접속 → 프로젝트 상태 확인
2. 서버가 꺼져 있으면 → `railway up --detach` 실행

### AI 응답만 안 돼요 (자동응답은 됨)
- API 크레딧 소진 가능성 → https://console.anthropic.com/settings/billing 확인
- 크레딧 충전하면 바로 복구됨

### 챗봇 디자인 바꾸고 싶어요
- `static/index.html` 수정 → 색상, 글씨, 버튼 등 변경 가능
- Claude Code에서 "챗봇 색상 파란색으로 바꿔줘" 같이 시키면 됨
